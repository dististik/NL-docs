Set Converter

This is a web-based converter to change Pok√©mon Showdown sets into the strings used by the Nisemon League strategy resource to make each set coppiable with buttons. All you need to do is paste a Showdown set into the text box below and click "Convert", however please be sure that the set follows the following criterea, otherwise the button will only return an error:

  • No whitespace between lines (unnecessary spaces at the end of lines)
  • Only one move/item/ability per slot
  • A space on both sides of each backslash seperating EV/IV values
  • Make sure to set the "Has IV Values" dropdown at the end of the converter to "True" if your set specifies IV values

If you recieve an error even while following the above criterea, let Alice know on Discord.
